HTML and CSS Quick-Reference Guide

HTML Tags

Header Tags

Header tags are, as its name suggests, used for headings. There are several sizes you can use for headings, from <h1> to <h6>, with <h1> being the biggest, and <h6> being the smallest.

Example: 

    <h1>This is a large heading!</h1>
    <h4>This is a smaller heading!</h4>
    <h6>This is the smallest heading!</h6>

Paragraph Tag

Commonly used to display blocks of text.

Example: 

    <p>
    Death is only meaningless if it does not change us, Vis.
    </p>

Acronym Tag

Used to add an acronym to text, which will display when the user hovers over the selected text.

Example: 

    <acronym title="The Way of Kings">TWOK</acronym>

Break Tag

Used to break up text, creating a new line.

Example: 

    <p>
    This is my first line.
    <br>
    This is my second line.
    </p>

Center Tag

A styling tag, which centers text nested within it.

Example: 

    <center>
    <p>
    My text is centered!
    </p>
    </center>

Marquee Tag

Creates a scrolling text area, which moves across the page. You can specify the direction.

Example: 

    <marquee>I'm scrolling right now.</marquee>

Image Tag

Allows you to insert an image.

Example: 

    <image src="C:\Users\user\Desktop\example.png">

Superscript Tag

Creates text a half-line above the bottom margin.

Example: 

    <sup>I'm small.</sup>

Subscript Tag

Creates text a half-line below the bottom margin.

Example: 

    <sup>I'm small but I'm on the bottom now.</sup>

Input Tag

Creates input fields with varying input types.

Example: 

    <input type="button"> I'm a button.
    <input type="checkbox"> Tick this.
    <input type="color"> Choose a color!


CSS Properties

Color Property

Changes text color.

Example: 

    p{
        color: blue;
    }

Background Color Property

Changes text color.

Example: 

    p{
        background-color: purple;
    }

Font Size Property

Changes the size of text

Example: 

    p{
        font-size: 20px;
    }

Padding Property

Adds space inside an element, between the border and contents inside the element.

Example: 

    .element{
        padding: 22px;
    }

Margin Property

Adds space outside the element, between this element and other elements.

Example: 

    .element{
        margin: 5px;
    }

Border Property

Creates a border around an element.

Example: 

    .element{
        border: 2px solid black;
    }

Text Align Property

Adjusts how text is aligned inside an element.

Example: 

    div{
        text-align: left;
    }

Position Property

Adjusts how an element is positioned.

Example: 

    .box{
        position: relative;
    }
